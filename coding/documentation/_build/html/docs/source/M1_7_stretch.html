M1_7_stretch module¶

M1_7_stretch.stretch(vector, n, plot=False)¶

Stretches Vectors to desired length by interpolating values

Example stretch([0,0,0,0,20,0,0,1],15)

with plotting stretch([0,0,0,0,20,0,0,1],15,True)

Parameters

  • vector (array(float)) – The input vector that should be stretched

  • n (int) – The length of the output vector after stretching

Returns

stretched vector

Return type

tensor
